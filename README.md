# Eurotrip Girls — GitHub Pages

Busca vuelos desde EZE hacia Europa una vez por día, guarda los resultados y envía alertas por Gmail.

## Secretos requeridos

- `SERPAPI_KEY`
- `EMAIL_USER`
- `EMAIL_APP_PASSWORD`
- `EMAIL_TO`

La ejecución manual está en **Actions → Buscar vuelos y publicar → Run workflow**.
